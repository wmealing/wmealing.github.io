The problem was that i had designed it as an [[https://adoptingerlang.org/docs/development/umbrella_projects/][umbrella project]], This type of application has multiple working
sub folders under the ./app/ folder.  I had only had one, with plans to add more related issues in the future.

Instead, I found out that others had been using my code, as is.. so I decided to restructure my app using the
rebar3 "app" template.

This should allow for applications to use this as a library, as therealcarloratm expected.


* Conclusion
:PROPERTIES:
:END:

My big takeaway here is to ensure that i dont use umbrella apps for libraries again.
